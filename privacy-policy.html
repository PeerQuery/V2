Privacy policy

All information we posses already exist on the Steem Blockchain.

Does Peer Query store or track sensitive user information?

All information stored through the use of our site already exist on the Blockchain. However, scanning the entire Blockchain everytime a user opens any page on Peer Query can be tidious, hence we only index some of the activities related to Peer Query so as to make easier to identify and access.

To make this indexing process efficient, we use a Database solution to index the following information about each new query created on Peer Query: author, permlink and time.

To be able to maintain an account of our users, we also register a list of all accounts that use our site. This information is what you see on our users page.

No keys, or passwords are required to use our site as login is processed by Steem Connect v2.

What about cookies?

Peer Query uses cookies to store information related to user sessions; nothing more.
