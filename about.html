Steem powered Query

Steem is a Blockchain which allows users to appreciate and reward content.

Peer Query is a query interface/client for Steem Condenser and makes it easier to browse and interact with queries created on Steem.

Handcrafted and maintained by Makafui George Dzivenu, founder of the Blockchain Rush; proudly Ghanaian.

Core technologies include: Steem, Steem.js, Steem Connect v2, Busy Img and Steem CMS by the Steem Institute.

Additional technologies include: Bootstrap 4, Node.js, Express.js, Font.Awesome and MySQL. See the source code on GitHub.


What is Peer Query?

Peer Query is a Steem client which allows users to create query types of posts to the Steem Blockchain.

How does Peer Query work?

Create a query on Peer Query and it will be published as a post to the Steem Blockchain. Steemians on all Steem clients can contribute and upvote your query, its as simple as that.

Does Peer Query cost anything to use?

Nope. There is no author reward splitting, no use of your voting power as payment and no direct or indirect charge to using Peer Query.

Who can use Peer Query?

Anyone with a Steem account.

How do I access my earnings on Peer Query?

Simply visit the wallet section of your account on Steemit to redeem your earnings. Currently we do not suport direct re-withdrawals on Peer Query.

Does Peer Query store or track sensitive user information?

Nope, read our Privacy Policy.

What types of queries can I create on Peer Query?

Currently we only support questions, but soon we will add support for more types of queries including polls, exchanges and more.
